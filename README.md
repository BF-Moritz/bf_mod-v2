# bf_mod-v2

Die 2. Version meines Twitch Chat Bots, jetzt auch komplett in Typescript.
Das UI wird noch komplett durch die [BF_Lib-Vue-v2](https://github.com/BF-Moritz/bf_lib-vue-v2) ersetzt.

---
